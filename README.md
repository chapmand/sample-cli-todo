# sample-cli-todo

A .NET 8 C# CLI to-do list app.

## Structure

- Models/TodoItem.cs
- Services/TodoService.cs
- Services/EmailService.cs
- Utils/ConsoleHelper.cs
- Data/todoitems.json

## Features
- List, create, update, delete, and complete to-dos
- JSON file persistence
- Email notifications
- CLI interface
